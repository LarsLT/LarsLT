# Hey, I'm Lars

I like building things that blink, spin and render. Most of my projects sit somewhere
between software and hardware: volumetric displays, smart mirrors, LED installations
and whatever else an ESP32 can be talked into doing.

Day to day I write mostly C++, Rust and Go. When something needs a web front I reach
for TypeScript and three.js. Right now I'm automating my 3D printer and building
tooling around it.

You can reach me at LarsTokkie@gmail.com.
